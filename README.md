# 刘宸乐5岁生日快乐 - 我的世界主题网站 🎉

一个精心设计的Minecraft主题生日庆祝网站，为刘宸乐小朋友的5岁生日打造！包含丰富的互动效果、游戏元素和惊喜彩蛋，特别优化移动端体验。

## 🎮 核心功能

### 🎯 互动游戏元素

#### 1. **点击挖方块小游戏** ⛏️
- 屏幕上随机生成可点击的我的世界方块
- 点击后方块破碎动画（像素破裂效果）
- 掉落生日祝福文字或分数
- 30秒限时挑战，得分超过50解锁成就
- 移动端触摸友好，按钮大小适中

#### 2. **建造生日蛋糕互动** 🎂
- 点击不同颜色方块来"建造"生日蛋糕
- 可以添加蜡烛装饰
- 完成后播放烟花庆祝动画
- 获得"蛋糕建造师"成就
- 支持重置重新建造

#### 3. **苦力怕倒计时彩蛋** 💥
- 生日日期倒计时显示
- 用苦力怕（Creeper）形象展示
- 点击苦力怕触发"嘶嘶"声和爆炸动画
- 倒计时接近零时苦力怕开始颤抖

### 🎨 视觉特效

#### 4. **3D方块漂浮动画**
- 背景中彩色方块3D旋转漂浮
- CSS 3D transforms 实现
- 移动端优化，减少方块数量提升性能

#### 5. **烟花粒子系统** ✨
- 页面加载时自动播放烟花
- 点击标题、照片按钮触发烟花
- Canvas 实现像素风格粒子效果
- 我的世界风格的像素烟花
- 每15秒自动触发一次

#### 6. **传送门照片效果** 🌀
- 紫色末地传送门粒子背景
- 照片可以像素化处理
- 传送门粒子实时动画
- 移动端滑动支持

### 🎵 音效系统

#### 7. **我的世界音效** 🔊
- 页面加载：传送门声
- 点击按钮：方块放置声
- 挖方块：方块破碎声
- 成就解锁：经验值获得声
- 苦力怕：特殊"嘶嘶"声效
- 村民：哼哼声
- 移动端大按钮控制音效开关（右上角）

### 🏆 成就系统

#### 8. **成就解锁弹窗**
- 访客浏览网站触发成就
- 🎉 「成就解锁：参加刘宸乐的5岁生日会！」
- 💥 「成就解锁：点击了苦力怕！」

- 🧑‍🌾 「成就解锁：和村民NPC互动了！」
- 我的世界风格的成就通知UI
- 使用 LocalStorage 记录本地成就（不需要服务器）

## 🚀 使用方法

3. 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
4. 最佳体验：Chrome/Edge 最新版本

### 添加照片

要添加生日照片：
1. 将照片文件命名为 `chenle.jpg` 并放入项目文件夹
2. 或在 `index.html` 中修改照片路径
